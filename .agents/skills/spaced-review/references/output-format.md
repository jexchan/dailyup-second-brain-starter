# Due Review Output

Use this format after listing due cards.

```markdown
# 今日复习

今天到期：N 张

1. [[card_name]] — 类型 / 优先级 / 上次复习 / 间隔
   - 主动提取：先不打开卡片，用 1–2 句话说出核心意思。
   - 评分：again / hard / good / easy
```

For a selected card, ask one retrieval question at a time. Do not reveal the
answer until the user has attempted retrieval.
