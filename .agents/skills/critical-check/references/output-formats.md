# Critical Check Output Formats

Choose the format selected in the main workflow.

## Quick Check

```markdown
## 快速判断
- **我对观点的理解**：...
- **最强的部分**：...
- **最大风险**：...
- **最该补的证据**：...
- **建议改写**：...
- **下一步验证**：...
```

## Full Check

```markdown
# 质疑与校验：{{内容标题 / 卡片名 / 主题}}

## 我对原观点的理解
{{2–4 句话复述核心主张}}

## 主张地图

| 类型 | 内容 | 证据状态 | 当前判断 |
|---|---|---|---|
| 中心主张 | {{主张}} | {{材料证据 / 外部证据 / 未核查}} | {{比较可靠 / 需要澄清 / 风险较高}} |
| 子主张 | {{主张}} | {{状态}} | {{判断}} |

## 最值得追问的地方

### {{追问标题}}
- **问题**：{{具体质疑}}
- **为什么重要**：{{它会影响什么结论}}
- **行动**：{{补证据 / 限定边界 / 找反例 / 改写 / 验证}}

## 证据与推理校验

| 主张 | 现有支撑 | 风险 | 下一步 |
|---|---|---|---|
| {{主张}} | {{证据/来源}} | {{风险}} | {{行动}} |
```
