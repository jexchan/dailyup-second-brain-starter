# Reading Coach Output Formats

Choose the smallest format that matches the user's request. Preserve the user's
language and actual learning work.

## Before-Reading Plan

```markdown
## 阅读使命
[one-sentence mission]

## 带着这 3 个问题读
1. ...
2. ...
3. ...

## 搜寻目标
- ...

## AI 的角色
[how AI should help while preserving productive struggle]
```

## ACTOR Learning Note

Use this only after the relevant ACTOR work has happened.

```markdown
## Aim｜我为什么读
[mission]

## Compress｜我应该带走什么
**Root:** ...

**Trunk:** ...

**Branches:**
1. ...
2. ...
3. ...

**Leaves worth keeping:**
- ...

## Test｜我需要质疑什么
- ...

## Own｜我如何用自己的话讲清楚
[the user's final teach-back]

**Connection to my life/work:**
- ...

**Gaps exposed and repaired:**
- ...

## Run｜我接下来怎么用
- Experiment or first action: ...
- Success criterion: ...
- AI may help with: ...
- AI steps back from: ...
```

## Reading-to-Action Output

Use this when the user mainly wants application.

```markdown
## 核心洞见
[one paragraph]

## 可以立刻改变的地方
- Decision: ...
- Rule: ...
- Checklist: ...
- Experiment: ...

## 风险与失效场景
- ...

## 复盘问题
1. ...
2. ...
3. ...
```
