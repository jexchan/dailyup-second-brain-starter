# Code of Conduct

## 我们的承诺

为了营造开放友善的协作环境，我们承诺无论年龄、体型、残障、族裔、性别、教育程度、经济状况、国籍、外貌、种族、宗教信仰或性取向如何，参与本项目的每个人都能获得无骚扰的体验。

## 我们的标准

**欢迎的行为：**

- 使用友善、包容的语言
- 尊重不同的观点和经历
- 有分歧时以建设性的方式沟通
- 关注对社区最有益的做法
- 对其他社区成员表达同理心

**不可接受的行为：**

- 使用性化的语言或图像
- 人身攻击、侮辱、嘲讽
- 公开或私下的骚扰
- 未经明确许可公开他人的私人信息
- 任何在专业场合不合适的行为

## 执行

项目维护者有权移除、编辑或拒绝不符合本行为准则的评论、commits、代码、wiki 编辑、issues 和其他贡献，并有权暂时或永久封禁不当行为的贡献者。

## 报告

如果你遇到或目睹不可接受的行为，请通过 GitHub Issues 私下反馈给维护者。所有举报会被认真、保密地处理。

## 来源

本准则改编自 [Contributor Covenant](https://www.contributor-covenant.org)，版本 2.1。
