# Obsidian

## 基本信息
| 项目 | 内容 |
|------|------|
| 名称 | Obsidian |
| 类型 | 笔记 / 知识管理 |
| 网址 | https://obsidian.md |
| 定价 | 个人免费；商业版付费；Sync / Publish 为付费服务 |
| 平台 | Mac / Windows / Linux / iOS / Android |

## 来源
Zettelkasten 与 PKM 社区最常推荐的本地笔记工具。

## 用途
基于本地 Markdown 文件的知识管理工具。特别适合构建"第二大脑"——卡片化、双向链接、长期积累。

## 核心功能
- **本地优先**：文件就是普通 Markdown，不锁定在云服务里
- **双向链接 + 图谱视图**：天然适配 Zettelkasten 的卡片网络
- **插件生态**：Dataview、Templater、Bases、Tasks 等扩展能力极强

## 为什么选择它
- 数据自己掌控（本地文件 + Git 版本控制）
- Markdown 纯文本，不用担心软件停运后内容丢失
- 社区模板/插件丰富，开箱即用的 PKM 工作流非常多

## 局限/缺点
- 上手稍陡峭（相比 Notion / Roam 的"开箱即用"）
- 多端同步需要自己搭（或付费买 Obsidian Sync）
- 默认的移动端编辑体验一般

## 替代方案
- Logseq（开源大纲式 + Git）
- Notion（云端数据库，协作好）
- Roam Research（双向链接鼻祖，但付费且闭源）

## 使用技巧
- 用 Git 管理 Vault → 永久版本历史
- 安装 Templater → 支持动态模板变量
- 安装 Dataview / Bases → 让笔记变成数据库可查询
- 快捷键：`Cmd+O` 快速跳转，`[[` 触发链接补全
