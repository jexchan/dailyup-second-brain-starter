# 课程设计五要素

## 适用场景
单节课程设计、整体课程体系规划

## 框架

```
┌─────────────────────────────────────────────────┐
│  1. 学习目标 (Learning Objectives)              │
│     学生能够做什么？                             │
│     用行为动词描述，而非"了解""理解"            │
├─────────────────────────────────────────────────┤
│  2. 核心内容 (Core Content)                     │
│     需要传递的知识、概念、技能                   │
│     与学习目标直接对应                           │
├─────────────────────────────────────────────────┤
│  3. 练习互动 (Practice/Interaction)            │
│     让学生动手或思考的活动                       │
│     检验理解、巩固记忆                           │
├─────────────────────────────────────────────────┤
│  4. 输出结果 (Output)                           │
│     完成学习后的具体产出                         │
│     可以是作品、决策、行动计划等                 │
├─────────────────────────────────────────────────┤
│  5. 课后延伸 (Extension)                        │
│     深入学习的方向                               │
│     与下一次学习的连接                           │
└─────────────────────────────────────────────────┘
```

## 使用要点

### 学习目标
- 使用具体的行为动词：能设计、能写出、能分析
- 避免：了解、理解、掌握（难以检验）
- 一节课的目标不超过 3 个

### 核心内容
- 与目标一一对应
- 先给图景，再展开细节
- 用例子和类比帮助理解

### 练习互动
- 不只是为了"活跃气氛"
- 练习应该直接检验学习目标
- 可以是思考、操作、讨论、创作

### 输出结果
- 让学习"看得见"
- 越具体的输出越好
- 可以作为后续学习的素材

### 课后延伸
- 给有兴趣的人指路
- 不要过度延伸，保持聚焦
- 可以预告下一次内容
