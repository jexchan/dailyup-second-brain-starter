# 问题-框架-落地

## 适用场景
方法论类文章、课程设计、知识型内容

## 结构

```
┌─────────────────────────────────────────────────────┐
│  1. 问题引入                                         │
│     - 提出读者关心的问题或痛点                        │
│     - 建立共鸣，说明为什么这个问题重要                │
├─────────────────────────────────────────────────────┤
│  2. 核心框架                                         │
│     - 给出整体图景                                   │
│     - 解释关键概念和关系                             │
│     - 用类比、例子帮助理解                           │
├─────────────────────────────────────────────────────┤
│  3. 落地方法                                         │
│     - 具体可执行的步骤                               │
│     - 常见误区与注意事项                             │
│     - 实际案例                                       │
├─────────────────────────────────────────────────────┤
│  4. 行动召唤                                         │
│     - 读完可以立即做的一件事                         │
│     - 进一步学习的方向                               │
└─────────────────────────────────────────────────────┘
```

## 使用要点
- 先回答"为什么要学"，再教"怎么用"
- 框架要简洁，3-5个核心要素为宜
- 每个环节都要有明确的 takeaway
- 结尾要有具体的行动建议
