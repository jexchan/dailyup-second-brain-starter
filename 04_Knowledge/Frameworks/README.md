# Frameworks 常用框架和方法论

这里存放常用的框架和方法论，用于指导和结构化思考。

## 框架列表

### 问题-框架-落地
通用内容结构框架，适用于文章、演讲、课程等内容创作。

### Learning OS 核心模块
面向 AI 时代的学习操作系统框架，包含 Input、Organize、Retrieval、Application、Output 五个模块。

### 课程设计五要素
课程设计的核心要素和指导原则。

## 如何使用

这些框架可以直接引用，也可以作为内容创作的结构参考。当需要：
- 设计一篇文章的结构
- 规划一个课程的框架
- 整理一个主题的知识

时，可以参考这些框架。
