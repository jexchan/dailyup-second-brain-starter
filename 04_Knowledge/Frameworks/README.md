# Frameworks 方法论手册

`Frameworks/` 只存放**可重复调用、需要整体阅读、能指导行动的方法论手册**。

边界规则：

- 单个概念、观点、模型 → 放入 `04_Knowledge/00_Cards/`
- 某个领域的系统学习过程 → 放入 `04_Knowledge/01_Topics/`
- 可作为完整流程反复调用的方法论手册 → 放入 `04_Knowledge/Frameworks/`

## 框架列表

### `_EXAMPLE_问题-框架-落地.md`

示例方法论手册：通用内容结构框架，适用于文章、演讲、课程等知识型内容创作。

## 如何使用

这些框架可以直接引用，也可以作为内容创作的结构参考。当需要：

- 设计一篇文章的结构
- 规划一个课程的框架
- 整理一个主题的知识

时，可以参考这些框架。

## 初始化清理

本目录中的 `_EXAMPLE_*.md` 文件是模板示例。使用 `scripts/init.sh` 初始化个人 Vault 时，会随其他示例文件一起清理。
