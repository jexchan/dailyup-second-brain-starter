# Core Principles

## 为什么重要
学习科学为 Learning OS 提供理论基础，帮助区分“感觉学到了”和“真的学会了”。

## 核心原则
### 1. 提取练习
真正有效的学习，不只是重看，而是努力把内容从大脑中提取出来。

### 2. 间隔重复
随着时间拉开复习间隔，长期保持效果更好。

### 3. 交错练习
不同类型问题或主题交替训练，有助于形成更强的辨别与迁移能力。

### 4. 认知负荷
学习设计要考虑大脑处理信息的负担，避免过载。

### 5. 输出促进理解
把内容讲出来、写出来、用出来，是学习的重要组成部分。
